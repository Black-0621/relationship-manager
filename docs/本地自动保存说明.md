# 本地自动保存说明

本版本已经把“浏览器缓存”改成“浏览器缓存 + 本地文件夹镜像”的双层保存。

## 使用方式

1. 用 `启动人际关系管理器.bat` 打开软件，地址会是 `http://127.0.0.1:4177/` 附近。
2. 进入“AI 与本地设置”。
3. 点击“关联/更换本地数据文件夹”。
4. 选择一个固定文件夹，例如：`D:\人际关系管理器数据`。
5. 之后每次保存人物、事件、聊天记录、设置、档案空间变化时，软件会自动写入：

```text
人际关系管理器完整数据.json
```

手动点击“立即保存到本地文件”时，还会在 `自动备份/` 下生成带时间戳的快照。

## 换浏览器/换电脑怎么恢复

1. 用新浏览器打开软件。
2. 进入“AI 与本地设置”。
3. 点击“关联/更换本地数据文件夹”。
4. 选择之前保存过 `人际关系管理器完整数据.json` 的文件夹。
5. 软件会提示是否读取旧数据，选择确定。

## 限制

浏览器安全规则不允许网页在未经授权的情况下直接写入任意本地文件夹，所以第一次必须由用户主动选择文件夹。之后在同一个浏览器里会尽量记住授权。

如果要做到完全像桌面软件一样固定写入 AppData，需要继续做 Tauri 桌面版，把当前网页应用嵌入桌面壳，并让 Rust 后端负责文件读写。
