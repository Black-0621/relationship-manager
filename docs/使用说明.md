# 人际关系管理器使用说明

## 1. 启动软件

双击 `启动人际关系管理器.bat`。

浏览器会打开本地地址，一般是：

```text
http://127.0.0.1:4177/
```

如果 4177 被占用，启动脚本可能会打开相邻端口。浏览器本地缓存按端口区分，所以正式使用时建议尽量固定使用同一个启动入口。

## 2. 建立人物档案

点击左侧“新建档案”，输入对象名称和当前阶段。

这个对象可以是朋友、暧昧对象、恋人、亲人、同事、合作伙伴，也可以是宠物。软件只负责记录和整理，不替你判断关系应该如何发展。

## 3. 记录事件

点击右上角“记录新事件”，写下发生日期、标题和详情。

建议优先记录事实，例如：

- 什么时候发生
- 谁主动联系
- 说了什么或做了什么
- 你当时的感受
- 后续有没有变化

事实越清楚，后续时间线和 AI 整理就越稳定。

## 4. 使用 AI 工作台

页面下方的 AI 工作台可以做三件事：

- 保存原文：把你粘贴的原始资料保存到当前人物档案里。
- 更新档案：让 AI 根据资料更新当前人物档案、阶段和关系维度。
- 和 AI 对话：围绕当前人物档案提问或补充信息。

“和 AI 对话”会保留聊天记录。关联本地文件夹后，聊天记录也会按日期保存成 Markdown。

## 5. 初建/重建故事线

如果你有一整段长资料，比如一年的交往经过、长期合作记录、宠物成长记录，可以把它粘贴进 AI 工作台，然后点击“初建/重建故事线”。

AI 会先生成预览。只有你确认后，它才会写入当前人物的时间线。

## 6. 关联本地保存文件夹

进入：

```text
AI 与本地设置 → 数据管理 → 关联/更换本地数据文件夹
```

选择一个你以后不会乱删的文件夹。关联后，软件会自动保存完整 JSON，并生成可读 Markdown。

核心数据文件是：

```text
人际关系管理器完整数据.json
```

可读记录文件夹是：

```text
人际关系管理器记录/
```

## 7. 换浏览器或换电脑

在新浏览器中打开软件后，重新关联之前那个本地保存文件夹。

如果文件夹里有 `人际关系管理器完整数据.json`，软件会提示是否恢复旧数据。

API Key 不会跟随数据文件恢复，需要在新浏览器里重新填写。

## 8. 导出可读档案

进入：

```text
AI 与本地设置 → 数据管理 → 导出可读档案
```

软件会为所有人物生成 Markdown 档案，适合人工阅读、长期备份或迁移。

## 9. 虚构测试材料

项目提供了单独的虚构测试材料：

```text
examples/虚构测试故事线.md
examples/虚构测试头像.png
```

它们不会被软件自动读取，也不会写进代码。需要测试 AI 整理效果或图片上传时，你可以自己手动上传或粘贴。
