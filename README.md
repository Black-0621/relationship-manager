# 人际关系管理器

人际关系管理器是一个本地优先的关系记录工具。它可以帮你记录某个人、某段关系、某个圈子里的日常互动，并把零散事件整理成时间线、关系阶段、关系维度和可读档案。

它最早只是一个玩笑式的小工具：我有一个女性朋友说，想管理自己的“鱼塘”，但手动记人、记聊天、记进展太麻烦，于是我给她做了一个“鱼塘管理器”。

后来我发现，这东西不只适合她，也可以给兄弟们用；再后来，它逐渐变成了一个更通用的关系记录工具。

你可以用它记录朋友、暧昧对象、亲人、同事、合作伙伴，也可以记录你和宠物之间的友好互动。

这个项目不鼓励操控关系，也不是情感裁判。它更像一个本地笔记本：帮你把发生过的事留好，把自己的判断和 AI 的整理分开，把事实、感受和分析放在一个更清楚的位置。

## 主要功能

- 人物档案：为不同对象建立独立档案，记录名称、标签、简介、人物画像和相处方式。
- 关系时间线：把一次次互动记录成事件，按日期排列，支持查看详情和删除。
- 关系阶段：为每段关系建立阶段节点，例如初识、熟悉、稳定、疏远、归档等。
- 关系维度：用熟悉度、信任感、好奇心、主动性、风险性等指标辅助观察当前阶段。
- AI 工作台：可以粘贴原始资料、对话或故事线，让 AI 帮你整理档案、重建时间线、分析当前关系。
- 聊天记录：和 AI 对话时，用户输入会立即进入聊天记录；关联文件夹后也会保存为 Markdown。
- 本地保存：默认保存在当前浏览器；关联本地文件夹后，会自动生成 JSON 数据镜像和可读 Markdown。
- 可读导出：一键导出所有人物的 Markdown 档案，方便人工阅读、备份或迁移。
- 多空间：可以建立多个本地档案空间，分别保存不同用途的数据。
- 主题与模式：可切换通用、亲情、职场、自定义等关系模式和配色。

## 直接使用

Windows 下双击：

```text
启动人际关系管理器.bat
```

它会用本地 Node 服务打开：

```text
http://127.0.0.1:4177/
```

建议使用 Chrome 或 Edge。首次运行前需要电脑已经安装 Node.js。

## 本地自动保存

进入：

```text
AI 与本地设置 → 数据管理 → 关联/更换本地数据文件夹
```

选择一个固定文件夹后，软件会在每次保存人物、事件、聊天记录、设置时自动写入：

```text
人际关系管理器完整数据.json
```

手动点击“立即保存到本地文件”时，会额外生成快照：

```text
自动备份/人际关系管理器快照-时间戳.json
```

可读 Markdown 会生成在：

```text
人际关系管理器记录/人物名/
```

其中会包含事件记录、原始资料、聊天记录和档案总览。

## AI 与隐私

- 不填写 API Key 时，AI 功能不会真正请求远程模型。
- API Key 只保存在当前浏览器本地，不会写入同步包和本地数据镜像。
- 点击 AI 功能时，软件才会把相关文字发送给你配置的模型接口。
- 上传照片会压缩后保存在本地数据里，不会自动发送给 AI。
- GitHub 发布版不包含真实用户资料、聊天记录、API Key 或本地备份数据。

```bash

```
